About This Dashboard

This dashboard demonstrates real-time data visualization from an embedded system. The custom interactive cursor adds a layer of sophistication to the minimalist interface. Data updates automatically every 5 seconds, with historical data points maintained for trend analysis.

In a production environment, this would connect to an actual ESP32 running a web server or MQTT broker. The Chart.js library provides elegant, performant data visualization without unnecessary complexity.

Integration Notes

To connect to an actual ESP32:

  • Replace /api/sensor-data with your ESP32 endpoint
  • Ensure CORS headers are properly configured on the ESP32
  • Implement JSON response format: { temperature, humidity, pressure, timestamp }
  • The dashboard automatically fallbacks to mock data if the API is unavailable
  • Data history is maintained for up to 20 points for chart visualization
